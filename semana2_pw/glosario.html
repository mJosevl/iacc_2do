Glosario de Conceptos Básicos de HTML

1. Etiquetas HTML

Las etiquetas HTML son elementos fundamentales para construir contenido en una página web. Estas etiquetas definen la estructura y el estilo del contenido que se muestra en un navegador web. Cada etiqueta tiene una apertura ("") y un cierre ("") que envuelven el contenido.

2. Hiperenlaces

Los hiperenlaces, también conocidos como enlaces o links, son elementos que permiten la navegación entre páginas web. Se crean mediante la etiqueta <a>. Para especificar la dirección del enlace, se utiliza el atributo href, que contiene la URL de destino.

Ejemplo: <a href="https://es.wikipedia.org/wiki/HTML">HTML en Wikipedia</a>.

3. Listas

Las listas organizan elementos relacionados en forma de ítems. HTML ofrece dos tipos principales: listas ordenadas (<ol>) y listas no ordenadas (<ul>). Cada ítem se define con la etiqueta <li>.

4. Tablas

Las tablas se utilizan para organizar datos en filas y columnas. Se crean con la etiqueta <table>, y las filas se definen con <tr> y las celdas con <td>.

Elemento HTML Descripción
<caption> Etiqueta opcional que proporciona un título o descripción para la tabla.
<colgroup> Etiqueta opcional para agrupar elementos <col> y aplicar estilos o atributos a columnas.
<thead> Etiqueta opcional que contiene filas de encabezados de tabla.
<tfoot> Etiqueta que contiene filas de resumen o datos de pie de tabla.
<tr> Fila en la tabla que contiene celdas de datos o encabezados.
<tbody> Grupo de filas que contiene los datos principales de la tabla.

5. Elementos Multimedia

Los elementos multimedia agregan interactividad y atractivo visual a las páginas web.

Imágenes (<img>): Se utilizan para mostrar imágenes en una página.

Audios (<audio>): Permite reproducir archivos de audio.

Videos (<video>): Se emplea para incrustar videos.
